otomi apply [options]

Description#

Apply Kubernetes resources.

OptionDescriptionValue TypeDefault
lHelmfile option to select charts by label (format: <label>=<value>), e.g. -l name=prometheus-operator or --label group=jobs[array]
-f, --fileHelmfile option to select helmfiles by filename, e.g. '-f helmfile.d/helmfile-15.ingress-core.yaml'[array]
--log-levelFATAL, ERROR, WARN, INFO, VERBOSE, DEBUG, TRACEWARN
-s, --skip-cleanup[boolean]false
-c, --set-context[boolean]false
-v, --verbosecount
--no-interactive, --ni[boolean]false
-h, --helpShow help[boolean]
